海淘帮助中心

什么是海淘？

简单的说，海淘就是去国外的购物网站买东西。

为什么需要海淘？

某些国外产品质量更令人放心
某些产品在国外的价格比国内便宜很多
某些产品在国内没的卖

哪些商品适合海淘？

适合海淘的商品
国内无出售的商品（如亚马逊Kindle）
与国内同款商品价格差别悬殊的商品（如婴童车、刮须刀、化妆品、保健品、名牌服装、奢侈品等）
某些国内不太令人放心的商品（如婴儿用品之奶粉、奶瓶等）
特价商品（如闪购网站Woot!和团购网站Groupon上推出的特价商品）

不适合海淘的商品
与国内同款商品差价太小的商品（如索尼数码相机）
体积大、质量重、运输不便且价值不高的商品
奇形怪状包装的、航空禁运（如压力罐）的商品
保鲜期短的食品（如巧克力）

海淘有哪些优势和劣势？

很多朋友认为，想买国外的东西也不一定要海淘，也可以采用网上代购和托去国外的朋友带的方式来买，但是实际上这三种方式是各有优劣的：

网上代购：优点在于省事，但是一般会收取10%左右的服务费。

托国外朋友买：
优点在于方便，只要跟朋友说好就可以了，
但是缺点有很多：
1. 国外实体店的东西不比网上的便宜。
2. 通常在国外的实体店消费都逃不掉消费税。
3. 托朋友千山万水带回来，非常麻烦，而你也欠下一个人情。
4. 可能朋友不大愿意接这种事，而碍于情面又不好说。

海淘：缺点在于过程相对麻烦，

优点在于购物过程灵活，有非常大的挑选的余地。全部过程都是自己完成，即享受了国外产品相对便宜的价格和优秀的质量，又不用去麻烦别人。
准备一张双币信用卡

在国外网站购物，一张有国际信用卡组织（Master/VISA/AE等）标记，且具有双币种结算功能信用卡是必须的。
