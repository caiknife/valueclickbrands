代码说明

文件说明

  • lib/settings.php 为配置文件，在这里配置您的ID和KEY
  • lib/oem.php 为获取数据文件
  • lib文件夹以外的文件都为是示例代码，您可以随意命名和更改
客齐集鼓励您自已作Cache，来提高用户的访问速度以及减少服务请求。

类（Class） Oem

version: 1.0
author: fbjiang@kijiji.com.cn

方法（Method）

  • function getService(string $service, array $options)

请求一个服务。
参数（Parameters）

$service string Values are:
  • adLoad 获取一条广告的内容。
  • adsLoad 获取多条广告内容。
  • categoryLoad 获取一个类目的信息。
  • categoryChildren 获取一个类目的所有子类目信息。
  • areaLoad 获取一个地区的信息。
  • areaChildren 获取一个地区的所有子地区信息。
  • searchLoad 获取一个搜索的结果。
$options array 不同的请求有不同的参数，详见服务请求对应的参数！

服务请求对应的参数

  • adLoad 获取一条广告的内容。
    代码 参数说明
    $oem->getService('adLoad ',array('id' => 14664343)); 'id'14664343为广告ID号。
    示例见：view.php

  • adsLoad 获取多条广告内容。
    代码 参数说明
    $oem->getService('adsLoad', array('categoryId' => 2104, 'areaId' => 21001, 'page' => 3)); 'categoryId'2104为类目ID号。
    'areaId'21001为地区ID号。
    'page'3为页数。
    示例见：listing.php

  • categoryLoad 获取一个类目的信息。
    代码 参数说明
    $oem->getService('categoryLoad', array('id' => 2104)); 'id'2104为类目ID号。
    示例见：cityhome.php

  • categoryChildren 获取一个类目的所有子类目信息。
    代码 参数说明
    $oem->getService('categoryChildren', array('id' => 2104)); 'id'2104为类目ID号。
    示例见：cityhome.php

  • areaLoad 获取一个地区的信息。
    代码 参数说明
    $oem->getService('areaLoad', array('id' => 21)); 'id'21为地区ID号。
    示例见：listing.php

  • areaChildren 获取一个地区的所有子地区信息。
    代码 参数说明
    $oem->getService('areaChildren', array('id' => 21)); 'id'21为地区ID号。
    示例见：index.php

  • searchLoad 获取一个搜索的结果。
    代码 参数说明
    getService('searchLoad', array('categoryId' => 2104 , 'query' => urlencode('兼职'), 'page' => 2)); 'categoryId'2104为类目ID号。
    'query'“兼职”为要搜索的关键词，关键词要用urlencode();。
    'page'2为页数。
    示例见：listing.php
