政策法规

给OEM之前的比较用心的站长一个更好的服务。好在什么地方：

  • 可以通过API获取客齐集网站内容
  • 可以建立完全自由控制的分类信息频道
  • 解决搜索引擎收录问题
  • 通过客齐集的通道快速申请Google Adsense成功
第一批满足以下条件可以加入
  • 已经参加客齐集频道定制服务
  • 发帖数在前30名的用户
其他用户可申请，请附带如下内容：
  • 您的网站和您的技术实力
  • 您准备如何推广OEM
哪些人可以推荐：
  • OEM premier帐户中，每季度发帖排名前三位的用户有一个推荐的名额
  • 客齐集员工可以推荐加入
参与规则：
  • 发帖必须使用客齐集的form
  • 通过客齐集注册Google Adsense，并使用
  • 网站上的其他广告信息不得有违法内容
规则：
  • 发帖数字在客齐集频道定制用户中排名前30名的用户，有资格可以加入OEM premier
  • 根据用户的技术实力和热情，选择第一批进入的用户
  • 加入的网站必须各个不同，有自己的创新内容，严禁复制
  • OEM premier末位淘汰制度，在3个月的时间内，发帖数字最低的3名，淘汰，增补新的用户进入
  • 您的ID和KEY不能泄露给其它人，如果泄露，将永久取消你的资格
  • 加入OEM premier后，你不得抄袭其它OEM premier用户的代码，情节严重者，将永久取消你的资格。
