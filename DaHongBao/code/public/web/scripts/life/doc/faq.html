常见问题

  • 我可以更改lib/oem.php吗？
    • 答：您可以根据您的实际需求，来自已更改。

  • 示例代码中的文件中的代码我可以自已改吗？
    • 答：客齐集只是提供一个可以运行的示例代码，您要改成什么样，可以自已来决定。但不要违反了相关的条款！

  • 我在网页上加客齐集的Adsense 这样我还能挣到钱吗？
    • 答：从客齐集注册Google Adsense 您会得到比正常申请多出很多的分成比例。

  • 我申请到的KEY是不变的吗？
    • 答：客齐集不保证您现在申请到的KEY是不变的！但在更改时客齐集会事先通知您的，会分配一个新的KEY给您的！再更改KEY的过程中，您只要更改一下配置文件就可以了。
