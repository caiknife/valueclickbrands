兑换奖品页面中罗列了我们为您准备的各色礼品。若您的红包数达到礼品的要求，便可以进行兑换。以下为兑换礼品的申请步骤。

1、点击您所看中的礼品兑换按钮：
   a、红包数不够。――>转到2
   b、若您的红包数足够。――>转到3

2、将有提示您还不能兑换此礼品。您可以通过“如何获得红包”来了解怎样能获得足够多的红包。

3、将进入资料确认页面，确认快递所必须的几项资料。注：若资料不属实，将无法获得礼品。
   a、资料正确――>转到4
   b、资料不正确――>转到5

4、点击提交按钮完成本次申请。

5、点击修改按钮进入个人资料编辑，对自己的资料进行编辑提交后，会自动转入资料确认页面，点击提交按钮完成本次申请。

工作人员会于每月的25至30日对上月25日到本月24日之间提交申请的人进行整理并以快递的方式发送礼品。而本月25日开始提出兑换申请的人则将于下个月进行整理发放。

每次发放我们都将在论坛“兑奖交流”板块登出名单以及兑换礼品名称和数量。若您在我们发出清单后很长一段时间内没有获得礼品，请在论坛上以发帖形式告知我们，我们将查询相关快递来确保您能获得该礼品。

大红包所有工作人员愿您在大红包能获得美好的网上体验。

大红包对此兑奖行为拥有最终解释权。
